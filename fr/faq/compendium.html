Qu'est ce qu'un Compendium ?

 

Installation

Pour importer un compendium d'un ami ou même pour le transférer, le moyen le plus rapide et simple est de :

  1. Aller dans le monde destination
  2. Créer dans votre monde un nouveau Compendium avec le même nom "compNom"
  3. Arrêter le monde
  4. Puis vous allez dans le dossier \Data\worlds de Foundry et cherchez votre world, il dois avoir le nom défini à sa création.
  5. Enfin vous allez dans le dossier \packs situé dans votre monde et remplacez le compNom.data par celui que vous avez récupérez (vous pouvez écraser l'ancien).
  6. Relancer le monde
